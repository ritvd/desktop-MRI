# desktop-MRI
dMRI is the first open-source simulator built to help teaching the concepts of MR physics.

| Package     | Version |
|-------------|---------|
| Python      | 3.10.6  |
| PySimpleGUI | 4.60.4  |
| NumPy       | 1.21.5  |
| OpenCV      | 4.5.4   |

## How to Use

Instructions on how this simulator can be used are present in instructions.docx.
